# IronLog

Offline-first workout tracking for Android.

## Download

Get the latest APK from the [Releases](https://github.com/DeccanHYD/Ironlog/releases) page.

## Features

- Fast set-by-set workout logging
- Training plans and built-in program templates
- Weekly volume analytics and recovery heatmap
- Body weight analytics and workout history
- Local-first data with backup and restore foundations

## Build From Source

```bash
npm install
npx expo run:android
```

If you need native folders generated locally first:

```bash
npx expo prebuild
```
