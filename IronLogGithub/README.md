# IronLog

IronLog is an offline-first workout tracking app built with React Native.

## Highlights
- Fast workout logging (sets, reps, weight)
- Volume analytics with radar and muscle breakdown
- Recovery heatmap (front/back body map)
- Program templates and plan editor
- Body weight analytics
- Local-first data with backup/restore foundations

## Android APK
You can download the latest APK from the GitHub Releases page.

## Tech Stack
- React Native
- Expo modules
- Android (Gradle)

## Local Build (Android)
```bash
cd android
gradlew.bat assembleRelease --console=plain
```

Release APK output:
`android/app/build/outputs/apk/release/app-release.apk`
