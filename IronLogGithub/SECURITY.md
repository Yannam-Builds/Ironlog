# Security Policy

## Reporting a Vulnerability

If you find a security issue, please do not open a public issue first.

Please report it privately with:
- clear description of the issue
- reproduction steps
- affected version/build
- potential impact

Temporary contact placeholder:
- Email: `security@ironlog.app`

## Response Targets
- Initial acknowledgment: within 72 hours
- Triage status update: within 7 days
- Patch timeline: depends on severity and reproducibility
