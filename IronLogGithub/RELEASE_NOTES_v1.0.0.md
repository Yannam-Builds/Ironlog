# IronLog v1.0.0

Initial public Android release of IronLog.

## What’s Included
- Workout logging with set-level tracking
- Plan builder and program templates
- Volume analytics with weekly radar and muscle breakdown
- Recovery heatmap (front/back body map)
- Body weight analytics
- Home program intelligence (history-aware, rule-driven recommendations)
- YouTube demo action from workout exercise menu
- Improved gesture smoothness and haptics behavior

## APK
- `app-release.apk` (Android)

## Known Limitations
- iOS build is not bundled in this release.
- Some advanced analytics and backup workflows are still evolving.

## Upgrade Notes
- This release is intended as the first public baseline.
- Existing local data should remain compatible with current storage schema.
