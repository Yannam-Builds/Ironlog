# Code of Conduct

## Our Pledge

We want IronLog to be a respectful, constructive project for everyone.

## Expected Behavior

- Be respectful and practical.
- Focus on the code and the problem.
- Assume good intent and communicate clearly.

## Unacceptable Behavior

- Harassment or personal attacks
- Discrimination
- Repeated disruptive behavior

## Reporting

Report conduct concerns privately to `conduct@ironlog.app`.
